Regression Analysis

Classifier: SVM versus Logistic Regression
SVM (Support Vector Machine) and Logistic Regression are supervised machine learning algorithms, primarily used for classification task. SVM finds an optimal hyperplane that separates the data into different classes while maximizing the margin. Logistic regression focuses on maximizing the probability of the class.
[Github]

Evaluation Metrics for Classification
When there is an unequal distribution of classes in the training dataset (biased/skewed distribution). It will be harder for the model to predict the minority class. If we have a test set with 100 patients (95 healthy and 5 cancer). If the model wrongly predicts all cancer patients still accuracy would be 95%. Therefore, accuracy is not a suitable evaluation metric for imbalanced classification.
[Github]

Bias Variance Tradeoff
The bias of the estimated function tells us the capacity of the model for correct predictions. The variance of the expected function tells the variation of the model predictions for new examples of the same distribution.
[Github]

Naive Bayes Classifier
Bayes' theorem calculates the probability of the event using prior knowledge and likelihood based on new information. The Naive Bayes classifier is based on Bayes' theorem.
[Github]

Ensemble Methods
Ensemble method is a technique that uses multiple machine learning models in order to obtain better performance. It includes reducing the variance, reducing the bias or increasing accuracy.
[Github]

Principal Component Analysis
It is a popular unsupervised technique to reduce the dimensionality/feature of the datasets. It preserves the maximum amount of information by finding new feature vectors that maximize the data spread.
[Github]

K-Mean Clustering
It is a unsupervied clustering algorithm where data points are assigned into k groups based on the closest distance between each group's centroid to the data point. All data points assigned to the same centroid form a group which is called a cluster. For k groups, we will have k centroid and corresponding k clusters.
[Github]
